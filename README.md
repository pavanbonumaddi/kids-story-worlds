# Kids Story World - PHP + MySQL Backend

## Features
- Attractive kids UI
- PHP + MySQL backend
- 144 sample stories
- Multi-language stories
- Telugu language support
- Language-wise text-to-speech
- Admin page to add new stories
- Category filter and search

## Files
- index.php
- admin.php
- config.php
- style.css
- script.js
- database.sql

## Setup in XAMPP
1. Copy this folder into `htdocs`.
2. Start Apache and MySQL.
3. Open phpMyAdmin.
4. Import `database.sql`.
5. Open:
   http://localhost/kids-story-world-backend/

## Admin Page
http://localhost/kids-story-world-backend/admin.php

## Telugu Voice Note
The website uses browser Text-to-Speech.
If the browser/system has Telugu voice support, Telugu stories will be read in Telugu.
Chrome/Edge works better.
